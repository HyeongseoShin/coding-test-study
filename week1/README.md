# 📘 1주차 문제집: 그래프 (BFS / DFS)

이번 주제는 **그래프 탐색 (BFS, DFS)** 입니다.  
아래의 공통 문제들을 풀이하고, 추가적으로 개인 문제 2개 이상을 풀어주세요.

## 📂 공통 문제

- **유형**: DFS, 백트래킹, 확률 계산

- **핵심 포인트**: 편의점 간 이동 가능 여부를 그래프로 모델링, 페스티벌 도달 여부 판단
